General Attributes

17 fields

Descriptive facts recorded on each system’s entry. These are free-form values rather than controlled options.

01 Description
A prose summary of what the system is and the workloads it targets.
02 Start Year
The year the project was first released or publicly announced.
03 End Year
The year active development ceased, if the project is discontinued.
04 History
Narrative of the system’s origins, milestones, and lineage over time.
05 Developer Organizations
The companies, labs, or communities that build and maintain it.
06 Country of Origin
Where the system was first developed.
07 System URL
The official homepage or product website.
08 Documentation URL
Where the official technical documentation is published.
09 Source Repository URL
The public version-control repository, where one exists.
10 Wikipedia URL
The system’s Wikipedia article, if available.
11 Twitter Handle
The project’s account on X / Twitter.
12 LinkedIn Handle
The project or vendor’s LinkedIn page.
13 Derived From
Systems whose codebase this one was forked or built from.
14 Embedded / Used By
Systems that embed or build upon this one as a component.
15 Inspired By
Systems that influenced this one’s design without shared code.
16 Compatible With
Systems whose wire protocol or dialect this one is compatible with.
17 Hosted Services (DBaaS)
Managed cloud offerings that run this system as a service.
